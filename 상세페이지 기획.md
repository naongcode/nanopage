# 📥 이커머스 상세페이지 시각화 프롬프트 생성 서비스 기획안 (Nano Banana Project)

## 1. 개요 및 목적

본 문서는 전문 이커머스 상세페이지 제작에 필요한 **15장의 이미지 촬영 시나리오(프롬프트)**를 사용자가 입력한 핵심 제품 정보와 마케팅 컨셉을 기반으로 자동 생성하는 웹 서비스의 기획안입니다.

**목표:** 콘텐츠 제작의 초기 단계인 '이미지 컨셉 기획'에 소요되는 시간과 비용을 절감하고, 이커머스 환경에 최적화된 스토리텔링 구조를 갖춘 구체적인 촬영 가이드라인(프롬프트)을 제공하여 제작 효율성을 극대화합니다.

## 2. 서비스 핵심 기능 및 구조

| 단계 | 기능명 | 설명 |
| :--- | :--- | :--- |
| **Step 1** | 제품 정보 입력 | 제품명, 카테고리, 경쟁사 컨셉, 타겟 고객 등 기본 정보를 입력받아 컨텍스트를 설정합니다. |
| **Step 2** | 마케팅 컨셉 도출 | 핵심 셀링 포인트 3가지, 주요 시각적 컨셉, 톤앤매너, 필수 연출 상황 2가지, 금지 요소 1가지를 입력받아 AI 프롬프트 생성의 핵심 변수를 확보합니다. |
| **Step 3** | 시나리오 자동 생성 | 입력된 정보를 기반으로, 이커머스 상세페이지의 효과적인 스토리텔링 순서(15컷)에 맞춰 **구체적인 촬영 시나리오(프롬프트)**를 AI가 생성합니다. |
| **Step 4** | 결과물 제공 | 생성된 15개의 시나리오를 역할(메인/모델/디테일) 및 페이지 목적과 함께 표 형태로 제공하며, `.md`, `.txt`, 또는 복사 가능한 형태로 제공합니다. |

## 3. 입력 필드 상세 정의 (Step 1 & 2)

사용자 인터페이스(UI)는 명확한 가이드라인과 함께 다음의 입력 필드를 제공해야 합니다.

### 3.1. 🛍️ 제품 기본 정보

| 입력 필드 | 예시 입력 | 필드 설명 및 가이드라인 |
| :--- | :--- | :--- |
| **제품명** | 프리미엄 무선 소음 차단 이어폰 | [텍스트 입력] |
| **카테고리** | 전자제품/음향기기 | 제품이 속한 대분류/소분류 |
| **차별화 컨셉** | 화려한 색상, 과격한 스포츠 활동 강조 | 경쟁사 제품과의 차별점을 부각하기 위한 정보 |
| **타겟 고객** | 조용한 환경에서 집중을 원하는 20~30대 직장인/학생 | 이미지 연출의 분위기와 모델 설정에 활용 |

### 3.2. 🎯 마케팅 컨셉 (AI 프롬프트 핵심 변수)

| 입력 필드 | 예시 입력 | 필드 설명 및 가이드라인 |
| :--- | :--- | :--- |
| **핵심 셀링 포인트 1** | 딥 노이즈 캔슬링 | 타겟 고객이 구매를 결정하게 만드는 핵심 요소 (특징 1) |
| **핵심 셀링 포인트 2** | 인체공학적 무중력 디자인 | 두 번째 중요한 특징 (특징 2) |
| **핵심 셀링 포인트 3** | 울트라 롱 배터리 | 세 번째 특징 (특징 3) |
| **주요 시각적 컨셉** | 고급스러운 사무 공간 / 자연광 홈카페 | 이미지의 배경과 분위기를 결정하는 핵심 컨셉 (1~2가지) |
| **톤앤매너/색상** | 차분한 뉴트럴 톤 (화이트, 베이지, 라이트 그레이) | 이미지의 전반적인 색감과 분위기 지침 |
| **필수 연출 상황 1** | 카페에서 노트북 작업을 하며 집중하는 모습 | 라이프스타일 컷 제작을 위한 구체적인 사용 상황 |
| **필수 연출 상황 2** | 출퇴근길 지하철/버스 안에서 휴식을 취하는 모습 | 두 번째 라이프스타일 연출 상황 |
| **금지 요소 1가지** | 과격한 스포츠 활동을 연상시키는 요소 | 경쟁사 컨셉이나 제품 이미지 훼손을 방지하기 위한 절대 금지 요소 |

## 4. 시나리오 출력 구조 및 역할 배분

총 15장의 이미지는 이커머스 상세페이지에서 가장 효과적인 스토리텔링 흐름에 맞춰 다음과 같이 3가지 유형으로 균형 있게 구성되어야 합니다.

| No. | 이미지 유형 | 역할 비중 | 페이지에서의 목적 (스토리 흐름) |
| :--- | :--- | :--- | :--- |
| **1~3** | 메인 (Main) | 20% | **도입 및 후킹:** 제품의 첫인상, 핵심 가치, 전체 분위기 제시. |
| **4~6** | 모델 (Model) | 20% | **착용 및 디자인:** 모델을 통한 제품의 크기, 핏, 스타일링 제시. |
| **7~9** | 디테일 (Detail) | 20% | **기능 강조:** 셀링 포인트 3가지를 시각적으로 증명. |
| **10~12**| 라이프스타일 (L/S) | 20% | **공감 및 사용성:** 필수 연출 상황과 일상에서의 자연스러운 사용 모습 제시. |
| **13~15**| 보조 (Sub) | 20% | **마무리 및 신뢰:** 다양한 각도, 비포/애프터 암시, 패키징 정보 제공. |

**AI 생성 프롬프트의 구체화 원칙:**

*   **배경/분위기:** 입력된 '주요 시각적 컨셉'과 '톤앤매너'를 문장 맨 앞에 명시하여 제작자에게 명확한 가이드라인을 제시.
*   **감성/묘사:** '차분한 뉴트럴 톤', '은은한 자연광', '무중력 암시' 등 감성적인 단어를 활용하여 시각적 연출의 디테일을 높임.
*   **기술적 연출:** 특징 1, 2, 3을 명확히 보여줄 수 있는 구도나 조명 활용을 지시합니다.

## 5. 상세 시나리오 예시 (15컷)

| No. | 이미지 유형 | 역할 | 구체적 시나리오 (프롬프트) |
| :--- | :--- | :--- | :--- |
| 1. | 메인 | 대표 이미지: 제품의 전체 디자인과 컨셉을 한 번에 보여줌. | [AI 생성 내용] |
| 2. | 메인 | 타겟 고객 공감: 제품의 특징 중 하나를 활용하여 고객의 니즈를 보여줌. | [AI 생성 내용] |
| 3. | 메인 | 감성 연출: 판매 페이지의 분위기를 고조시키는 감성적인 배경 컷. | [AI 생성 내용] |
| 4. | 모델 | 착용컷 1: 필수 연출 상황 1을 구현하는 여성/남성 모델의 착용컷. | [AI 생성 내용] |
| 5. | 모델 | 착용컷 2: 필수 연출 상황 2를 구현하는 다른 성별/인종 모델의 착용컷. | [AI 생성 내용] |
| 6. | 모델 | 디자인 강조: 제품의 전체 디자인을 강조하는 전신 착용 모델 컷. | [AI 생성 내용] |
| 7. | 디테일 | 특징 1 강조: 가장 중요한 특징 1을 클로즈업. | [AI 생성 내용] |
| 8. | 디테일 | 특징 2 강조: 두 번째 중요한 특징 2를 클로즈업. | [AI 생성 내용] |
| 9. | 디테일 | 특징 3 강조: 세 번째 특징 3의 사용 장면을 클로즈업. | [AI 생성 내용] |
| 10. | 라이프스타일 | 휴식/릴렉스: 편안한 환경에서 제품이 자연스럽게 놓여있는 컷. | [AI 생성 내용] |
| 11. | 라이프스타일 | 활동적인 장면: 제품이 활발한 활동 중에도 사용됨을 보여주는 컷. | [AI 생성 내용] |
| 12. | 라이프스타일 | 제품 단독 컷: 배경이 없는 흰색 배경 위에 그림자만 있는 순수한 제품 컷. | [AI 생성 내용] |
| 13. | 모델 | 다양한 각도: 제품의 다른 각도나 뒷모습 등을 보여주는 모델 컷. | [AI 생성 내용] |
| 14. | 메인 | 사용 전후 대비 (암시): 제품 사용으로 얻는 이점을 암시하는 비포/애프터식 연출. | [AI 생성 내용] |
| 15. | 디테일 | 패키징 또는 구성품: 제품 패키징이나 함께 제공되는 구성품을 보여주는 컷. | [AI 생성 내용] |

## 6. 향후 확장 가능성

*   **AI 이미지 생성 연동:** 생성된 프롬프트를 바로 Midjourney나 DALL-E API에 연결하여 썸네일 예시 이미지를 1차적으로 제공.
*   **스토리보드 기능:** 생성된 15컷의 시나리오를 드래그 앤 드롭으로 순서를 변경할 수 있는 기능 추가.
*   **제품 카테고리별 특화 템플릿:** 의류, 식품, 화장품 등 카테고리별로 최적화된 15컷의 역할 배분 템플릿 제공.
